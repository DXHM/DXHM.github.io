此项目是基于 Rust 编写的密码破解框架。

集成多种的密码破解功能，包括字典攻击、弱口令检测和哈希破解等功能。

etc..

Repository

Github：BlackCracker_Rusty
最新版本下载：Release

Functionality

  • 字典攻击模式：通过遍历字典文件中的密码，尝试破解目标密码。
  • 弱口令检测模式：检测目标密码是否为弱口令，进行口令安全性评估。
  • 哈希破解模式：通过破解密码哈希值，尝试还原原始密码。
