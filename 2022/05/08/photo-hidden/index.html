  • Steghide

Steghide是一个可以将文件隐藏到图片或音频中的工具

安装
apt-get install steghide

使用
steghide –help #查看帮助

查看图片中嵌入的文件信息
steghide info 1.jpg

提取图片中隐藏的文件
steghide extract -sf 1.jpg
